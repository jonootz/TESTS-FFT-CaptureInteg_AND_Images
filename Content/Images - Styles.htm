  UI Controls
  Name: Name of the class (If left blank, changes will be made to the element's main style)
  Comment: Lets you add a comment to the style class.
  Properties for the New Class: All columns should be sortable and not resizable.
  Columns
  Include: Allows for the selection of specific properties to include or exclude from the new style or style update
  Property: Displays the CSS property of the class to be updated
  Value: Displays the value of the property to be updated
  Select Stylesheet to modify: Allows for the selection of the stylesheet to which the new class/style will be added. If the topic is associated with multiple stylesheets, this column should be sortable.
  Create style and update the source element: Creates the style/class and applies it to the selected element
  Create style without updating the source element: Creates the style/class but doesn’t apply it to the selected element
