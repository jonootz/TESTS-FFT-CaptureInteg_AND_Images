  Images - File Location & Properties
 

Images - Location in Project
  Image files can exist anywhere within the Flare project's Content folder, however by default, Image files are saved to the Content/Resources/Images folder.

  Right-click menu items (Tooltips in parentheses):
  Preview (Preview file with different condition tag expressions): Opens a Preview of the selected content file (Only active for topic files).
  Open (Open): Opens the selected Content file in the MDI.
  Open with: Gives options to open the Content file in the Internal Text Editor, Internal Browser and other applications.
  Source Control: Invokes the Source Control sub-menu (covered in the Source Control Test Plan).
  Cut (Cut): Cuts the selected Content to the clipboard
  Copy (Copy): Copies the selected Content file to the clipboard.
  Paste (Paste): Pastes the Content file from the clipboard
  Delete (Delete): Removes the selected Content file from the project.
  Rename (Rename): Allows you to rename the Content file
  Edit with MadCap Capture (Edit this file with MadCap Capture): Opens the selected image file in MadCap Capture. This option is only present when right-clicking an image file.
  Regenerate Captured Image (Regenerate Captured Image): Regenerates an Image to remove a watermark or regenerates a PROPS file. Only appears when an image is right clicked.
  Convert to XML: (Only active for non-XML topic files) Invokes the HTML -> XHTML Conversion dialog which lets you convert the non-XML topic file to XML
  Send to Folder… (Send to Folder): Invokes the Send to Folder dialog which lets you copy the file out of the project to any location,
  Properties (Display properties for the selected item): Opens the Properties dialog window for the selected file.
  View Links (Open this file in the Link Viewer): Opens this file in the Link Viewer.
  Locate in TOC (Locate file in TOC(s)): Opens the TOC Editor and locates the content file in the TOC. If the content file occurs in multiple TOCs, this will open an untitled dialog which lets you select between all TOCs in the project which link to the content file.
 

(Image) Properties dialog
  To open:
  Right-click the Image in the Content Explorer and select Properties from the context menu.
  Select the Image in the Content Explorer and press F4.
  With the Image open and in focus in the MDI, press CTRL+Shift+P on the keyboard
  Select the Image in the Content Explorer and click the Properties button in the Local Toolbar (of the Content Explorer).
  Toolstrip interface:
  Select the Image in the Content Explorer and select !Edit | Properties.
  Select the Image in the Content Explorer and click the Properties button in the Standard Toolbar.
  With the Image open and in focus in the MDI, go !File | Properties
  Ribbon interface:
  Select the Image in the Content Explorer and select !Home | Properties.
  With the Image open and in focus in the MDI, go !File | Properties
